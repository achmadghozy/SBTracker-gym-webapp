You're senior engineer with specialities in mobile progressive web application
you want to design a system that can be used to track the progress of a user's fitness journey
User story:
As a user, I want to be able to track my fitness journey so that I can see my progress and stay motivated.
I want to take record of my workout sessions, including the exercises I performed, the weight I lifted, and the number of reps I completed easily in the front page of application, and before I exercise I want to be able to see my workout plan for the day.
I can plan my workout easily by selecting plan that I want to follow, and the application will generate a workout plan for me based on my selection. Workout plan is a weekly plan with list of exercises with the number of sets and reps for each exercise.
